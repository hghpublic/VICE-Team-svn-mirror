=== build

[source, shell]
----
cd vice
./autogen.sh > dev/logs/autogen.sh.log 2>&1
./configure > dev/logs/configure.log 2>&1
make > dev/logs/make.log 2>&1
----

==== `cmake-bootstrap.sh`

[source, shell]
----
./cmake-bootstrap.sh > dev/logs/cmake-bootstrap.sh.log 2>&1
----

==== configure help

[source, shell]
----
./configure --help > dev/configure-help.txt 2>&1
----
